# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest  | ✅ Yes    |

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not open a public issue**.

Instead, send a private email to the maintainer at the address listed in the
commit history or reach out via the project's GitHub Discussions.

You should receive a response within 48 hours. If you don't, please follow up.

Do not share details of the vulnerability publicly until a fix has been
released and announced.
